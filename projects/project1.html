Pipeline to C++ and OpenGL

After completing the 3D Mask, I exported the vertexes of the file using a Python script ran directly in Blender ( here's the link to the file ). The script writes the data directly into a C++ file included in my main.cpp with a header (in which I made the data set extern , so that it can be accessed, used and edited everywhere).
From there, I could use OpenGL to display the mask in a window as follows:

Animation

I made all the phonemes (mouth shapes associated with a specific sound) in Blender. Following this, I exported them all to my C++ project.
Finally, I displayed a set of vertices that I interpolated from one phoneme to another one. Here's a sample of the transition from neutral face to "a" phoneme:

Speaking

Now, we can animate a succession of phonemes, let's give sense to it. We want to animate sentences, and any sentence.
So I made a dictionnary/map that associates a sequence of letters to a phoneme id number (cf next picture).

After improving the interpolating system in order to be able to do it successively. And after making algorithms to manipulate strings to turn them into sequences of phonemes id numbers:
I could finally develop the feature; make the mask animated to visually pronounce the user's sentence input.

Here's a recording of the program. (The translating of texts into Phonemes is based on french pronounciation, so the examples are french sentences).
